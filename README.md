# AiDongLai

爱东来，自油爱。
一个基于Android无障碍服务的自动化工具。

## 功能特点

- 自动化店铺搜索和进入
- 简单易用的界面
- 完全开源免费

## 使用说明

1. 安装应用并打开
2. 开启无障碍服务（出现黑框表示开启成功）
3. 使用步骤：
   - 进入某音 -> 我 -> 商城 -> 搜索
   - 输入想要的店名
   - 切换到店铺栏
   - 进入店铺即可完成
